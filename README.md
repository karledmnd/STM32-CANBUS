# STM32-CANBUS
A custom PCB project based on STM32F302C6T6 microchip that can communicate with a car's CAN network.

Software used:
* EasyEDA
* STM32CubeIDE

Hardware & tools used:
* ST-Link V2 Programmer
* MacBook Air M1
* A handful of jumper wires
* PCB stencil
* Soldering station
* Hot plate
* Misc. soldering items (paste, wick, etc.)
